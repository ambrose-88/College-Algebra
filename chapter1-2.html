    Section 1.2 Graphing Points on the Coordinate Plane

    In the last section, we reviewed terminology related to coordinate planes. In this section, we will demonstrate how to graph points. We will draw points by hand, and we will use technology to plot points. We will build upon this skill when we learn how to graph equations and functions.

    Subsection 1.2.1 Graphing a Point by Hand

    Subsubsection 1.2.1.1 Positive and Negative Signs as Directions of Travel.

    When graphing on a number line, all the numbers to the right of 0 are positive, and all the numbers to the left of 0 are negative. To understand why we label the number line this way, think of a car parked at \(x=0\) on an infinite straight road. \(x=0\) is the starting point (also known as the reference point); it is where the car is located before we start measuring how far it travels. The positive numbers indicate how many miles forward the car travels from the starting point; the negative numbers indicate how many miles backward the car travels in reverse from the starting point.
    Unlike a number line, the coordinate plane uses two number linesone horizontal and one vertical. Each axis has a positive (forward) and negative (backward) direction. On a standard coordinate plane, moving to the right (east) is the positive horizontal direction, and moving up (north) is the positive vertical direction. An ordered pair \((x,y)\) tells you how far and in which direction to travel along each axis. The number tells you how far to travel, and the sign of the number tells you which direction to travel. For example, the point \((-3,2)\) means thatstarting at the originyou need to move 3 units to the left and 2 units up.

    Subsubsection 1.2.1.2 Graphing a Point By Hand

    When we look at our rectangle example for \((3,4)\) from the last section, notice that there were two ways we could travel the perimeter of the rectangle to reach the point. We could travel 3 along the south edge, and then go up by 4 along the east edge. This is the standard way to plot the point. Equivalently, we could have gone up 4 along the west edge, then traveled 3 along the north edge. While technically not the standard method, there is no harm in plotting a point this way. It can be shown any point on the rectangular coordinate plane can be reached using either method.
    Procedure 1.2.4. How to Graph a Point.
    Let \((x,y)\) be any point on the rectangular coordinate system. To plot the point, you can either
    1. Travel horizontally by \(x\text{,}\) then move vertically by \(y\text{,}\) or
    2. Travel vertically by \(y\text{,}\) then move horizontally by \(x\text{.}\)
    The following set of examples demonstrates how to graph points by hand in all four quadrants of the coordinate plane and on the axes.
    Example 1.2.5. Plotting in Quadrant 1.
    Plot the point \((2,4)\text{.}\)
    Solution.
    Start at the origin, move 2 to the right, and then up by 4.
    Example 1.2.7. Plotting in Quadrant 2.
    Plot the point \((-1,3)\text{.}\)
    Solution.
    Start at the origin, move 1 to the left, and then up by 3.
    Example 1.2.9. Plotting in Quadrant 3.
    Plot the point \((-3,-2)\text{.}\)
    Solution.
    Start at the origin, move 3 to the left, and then down by 2.
    Example 1.2.11. Plotting in Quadrant 4.
    Plot the point \((4,-4)\text{.}\)
    Solution.
    Start at the origin, move 4 to the right, and then down by 4.
    Example 1.2.13. Plotting on the \(x\)-axis.
    Plot the point \((3,0)\text{.}\)
    Solution.
    Start at the origin, move 3 to the right, and then stay on the \(x\)-axis.
    Example 1.2.15. Plotting on the \(y\)-axis.
    Plot the point \((0,-2)\text{.}\)
    Solution.
    Start at the origin, stay on the \(y\)-axis, and then move down by 2.

    Subsubsection 1.2.1.3 Graphing Points Using Technology

    In this course, we will use the Desmos as our graphing calculator. (An app version of Desmos is also available for both Android and iOS.) If you prefer to use a TI-84, you may visit this page to see how to graph points on a TI-84.
    Graphing points in Desmos is simple! Type the point into the Input Field, and it will automatically plot it for you. Let replot our points from our last six examples all at once in Desmos.
    Example 1.2.17. Plotting Points in Desmos.
    Plot the points \((2,4)\text{,}\) \((-1,3)\text{,}\) \((-3,-2)\text{,}\) \((4,-4)\text{,}\) \((3,0)\text{,}\) and \((0,-2)\) in Desmos.
    Solution 1.
    Type each point on its own separate line in the Input Panel, and the points will automatically plot on the graph.
    Solution 2.
    You can write all the points on a single line with list notation. To create a list of points, type a square left bracket [, a comma separated list of points, and then close the list with a right square bracket ]. If you have it typed correctly, it should look like this:
    \begin{equation*} [(2,4), (-2,3), (-3,-2), (4,-4), (3,0), (0, -2)]\]. \end{equation*}
    The output will render in Desmos like this.

    Subsubsection 1.2.1.4 Using Variables Other Than \(x\) and \(y\text{.}\)

    While using \(x\) and \(y\) is common in mathematics, when working with real-world examples, you may want to use variables to represent the quantities in the problem. Informally, we can even use words or short phrases to represent \(x\) and \(y\text{.}\) This is commonly used to label charts and reports.
    The rectangular coordinate plane is also called the \(xy\)-plane to emphasize which variable names we are using. When using variable names other than \(x\) and \(y\text{,}\) we will use the >\(xy\)-plane naming convention, but substitute the variables relevant to the problem context.
    Example 1.2.20. Economics.
    If \(x\) is the quantity of items produced by a factory and \(y\) is the cost (in thousands of dollars) to produce those items, we could use \(q\) instead of \(x\) for quantity produced and \(C\) instead of \(y\) for the cost to produce \(q\) items. Ordered pairs would be \((q,C)\) plotted in the \(qC\)-plane.
    Example 1.2.22. Physics.
    In the study of motion, it is common to represent time and position using variables. While mathematics often uses \(x\) for time and \(y\) for position, physicists typically use \(t\) to denote time and \(s\) to represent the position of an object relative to its starting point. Thus, the motion of an object can be described using ordered pairs \((t, s)\text{,}\) which are plotted in the \(ts\)-plane.
    Example 1.2.24. Biology.
    In a biology experiment, let \(d\) represent the number of days since a seed was planted, and let \(h\) be the height of the plant (in centimeters) after \(d\) days. Ordered pairs would be \((d, h)\text{,}\) plotted in the \(dh\)-plane.
    Now that you have seen several examples of how to use other variables, lets try do a full example from psychology.
    Example 1.2.26. Psychology.
    In a psychology study, researchers recorded data about participants daily screen time and reported stress levels. Let \(x\) represent the average number of hours of screen time per day, and let \(y\) represent the participants stress level on a scale from 1 to 10.
    • What variable names might better represent these quantities than using \(x\) and \(y\text{?}\) What would the ordered pairs be instead of \((x, y)\text{?}\) What would we call the \(xy\)-plane when we use these variables?
    • Suppose the study reports the point \((5, 8)\text{.}\) Interpret this point in the context of the study.
    • One participant reported using screens for 2.5 hours in a day and had a stress level of 4. Write this information as a coordinate point. Then graph the point on the same coordinate plane as \((5, 8)\text{.}\)
    Solution.
    • A better variable name for average daily screen time could be \(h\) for screen time in hours, and a better variable name for stress level could be \(S\) for stress level. The ordered pairs would be \((h,S)\text{,}\) and we would call the \(hS\)-plane the screen time-stress level plane.
    • The point \((5, 8)\) means that \(h = 5\) and \(S = 8\text{.}\) Therefore, a participant who used screens for an average of 5 hours per day reported a stress level of 8 on the 110 scale.
    • Since the participant reported an average screen time of 2.5 hours, we have that \(h = 2.5\text{.}\) Their stress level is a 4; therefore, \(S = 4\text{.}\) Since our points are of the form \((h, S)\text{,}\) we should use the ordered pair \((2.5, 4)\) to represent this individual.
