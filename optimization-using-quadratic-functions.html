    Section 4.15 Optimization Using Quadratic Functions

    Optimization is an area of mathematics specializing in finding the absolute extrema of an equation to solve a real-world problem. Quadratic functions give a solid introduction to the topic with many mathematical and real-world applications.

    Subsection 4.15.1 Applications to Geometry

    Example 4.15.1. Minimizing the Distance from a Point to a Parabola.

    Find the minimum distance between the parabola \(y = x^2\) and the point \((0, 3)\text{.}\)
    Answer.
    The minimum distance is \(\frac{\sqrt{11}}{2}\text{.}\)
    Solution.
    Let \((x, x^2)\) be a point on the parabola. The distance between this point and \((0, 3)\) is given by
    \begin{equation*} D = \sqrt{(x - 0)^2 + (x^2 - 3)^2}. \end{equation*}
    Since distance is always positive, minimizing \(D\) is equivalent to minimizing \(D^2.\) Therefore, we need to minimize
    \begin{equation*} D^2 = (x-0)^2 + (x^2 - 3)^2. \end{equation*}
    Simplifying the equation, we get
    \begin{align*} D^2\amp= (x-0)^2 + (x^2 - 3)^2 \\\\ D^2\amp= x^2 + (x^4 - 6x^2 + 9) \\\\ D^2\amp= x^4 - 5x^2 + 9 \end{align*}
    Notice that this equation is quadratic in form. Let \(u = x^2\) and \(u^2=(x^2)^2=x^4\text{.}\) This implies that
    \begin{equation*} D^2 = u^2 - 5u + 9. \end{equation*}
    Since this is a quadratic function with \(a >0\text{,}\) we know that the vertex is both a local and absolute minimum. Therefore, the value of \(k\) is the minimum value of \(D^2\text{.}\) First, we have to find the value of \(h\text{:}\)
    \begin{align*} h\amp= -\frac{b}{2a} \\\\ \amp= -\frac{-5}{2(1)} \\\\ \amp= \frac{5}{2} \end{align*}
    Now, we let \(u = h=\frac{5}{2}\) and substitute into the equation for \(D^2\) to find the value of \(k\text{:}\)
    \begin{align*} k\amp= \left(\frac{5}{2}\right)^2 - 5\left(\frac{5}{2}\right) + 9 \\\\ \amp= \frac{25}{4} - \frac{50}{4} + \frac{36}{4} \\\\ \amp= \frac{11}{4}\\ \end{align*}
    Thus, the minimum value of \(D^2\) is \(\frac{11}{4}\text{,}\) and the minimum distance is
    \begin{align*} D\amp= \sqrt{D^2} \\\\ D\amp= \sqrt{\frac{11}{4}} \\\\ D\amp= \frac{\sqrt{11}}{\sqrt{4}} \\\\ \amp= \frac{\sqrt{11}}{2} \end{align*}

    Subsection 4.15.2 Applications from Physics

    Theorem 4.15.3. Height of Projectiles.

    The height of a projectile (in feet, ignoring air resistance) is given by the equation
    \begin{equation*} h(t) = -16t^2 + v_0t + h_0, \end{equation*}
    where \(v_0\) is the initial velocity (in feet per second) and \(h_0\) is the initial height (in feet).

    Example 4.15.4. Finding the Maximum Height of a Projectile.

    A soccer ball is kicked upward with an initial velocity of \(40\) ft/s from a height of \(3\) feet. Its height after \(t\) seconds is given by \(h(t) = -16t^2 + 40t + 3.\) Find the time when the ball reaches its maximum height and determine that height.
    Answer.
    The ball reaches a maximum height of \(28\) feet after \(1.25\) seconds.
    Solution.
    The height function \(h(t) = -16t^2 + 40t + 3\) is a downward-opening parabola. The time of the maximum height occurs at the vertex, given by
    \begin{align*} t\amp= -\frac{b}{2a} \\\\ \amp= -\frac{40}{2(-16)} \\\\ \amp= 1.25 \text{ seconds.} \end{align*}
    Substitute \(t = 1.25\) into the height equation to find the maximum height:
    \begin{align*} h(1.25)\amp= -16(1.25)^2 + 40(1.25) + 3 \\\\ \amp= -16(1.5625) + 50 + 3 \\\\ \amp= -25 + 50 + 3 \\\\ \amp= 28 \text{ feet.} \end{align*}

    Example 4.15.6. Finding Initial Height and Velocity from Maximum Height.

    Suppose a rocket reaches a maximal height of \(700\) feet, \(5\) seconds after launch. Find its initial velocity and initial height.
    Answer.
    The rocket was launched from a height of \(300\) feet with an initial velocity of \(160\) ft/s.
    Solution.
    Since the maximal height occurs at the vertex of the height equation \(h(t) = -16t^2 + v_0t + h_0\text{,}\) we know that the vertex is at the point \((5,700)\text{.}\) In the height function, notice that \(b=v_0\text{.}\) Since we know the values of \(t\) and \(a\text{,}\) we can solve for \(v_0\) using the formula \(t=\frac{-b}{2a}.\) Plugging these values in, the following calculations shows us how to find \(v_0\text{:}\)
    \begin{align*} 5\amp= -\frac{v_0}{2(-16)} \\\\ 5\amp= \frac{v_0}{32} \\\\ v_0\amp= 5(32) \\\\ v_0\amp= 160 \text{ ft/s.} \end{align*}
    Now that we know the initial velocity, we can substitute \(t=5\text{,}\) \(v_0=160\text{,}\) and \(h(t)=700\) into the height equation to find the initial height \(h_0\) The following calculation shows us how do to this:
    \begin{align*} 700\amp= -16(5)^2 + 160(5) + h_0 \\\\ 700\amp- -16(25) + 800 + h_0 \\\\ 700\amp= -400 + 800 + h_0 \\\\ 700\amp= 400 + h_0 \\\\ h_0\amp= 700 - 400 \\\\ h_0\amp= 300 \text{ feet.} \end{align*}
    Therefore, the equation of the height function is
    \begin{equation*} h(t) = -16t^2 + 160t + 300. \end{equation*}

    Subsection 4.15.3 Applications from Economics

    Example 4.15.7. Maximizing Revenue in Game Design.

    A mobile game sells for \(\$1.00\) and attracts \(50{,}000\) downloads. For every \(\$0.10\) increase in price, downloads decrease by \(2{,}000\text{.}\) Find the maximum revenue, the price that maximizes revenue, and the number of downloads at that price.
    Answer.
    The price that maximizes revenue is \(\$1.75\text{,}\) yielding approximately \(35{,}000\) downloads. The maximum revenue is \(\$61{,}250.\)
    Solution.
    Let \(x\) represent the number of \(\$0.10\) increases. Then the price is \(p = 1 + 0.1x\text{,}\) and the number of downloads is \(n = 50{,}000 - 2{,}000x\text{.}\)
    The revenue \(R\) is the price of the video game times the number of downloads. Converting that into symbols and simplifying, we get the following calculation:
    \begin{align*} R\amp= p \times n \\\\ \amp= (1 + 0.1x)(50{,}000 - 2{,}000x) \\\\ \amp= 50{,}000 + 5{,}000x - 2{,}000x - 200x^2 \\\\ \amp= -200x^2 + 3{,}000x + 50{,}000 \end{align*}
    Since the value of \(a\) is negative, the parabola opens downward, and the \(R\)-coordinate of the vertex is the maximum profit. To find the value of \(R\) at the vertex, we first find the value of \(x\) at the vertex using the formula \(x = \frac{-b}{2a}.\) The following calculation shows how to do this:
    \begin{align*} x\amp= \frac{-b}{2a} \\\\ \amp= \frac{-3{,}000}{2(-200)} \\\\ \amp= 7.5 \end{align*}
    Now, we substitute \(x = 7.5\) into the revenue equation to find the maximum revenue:
    \begin{align*} R(7.5)\amp= -200(7.5)^2 + 3{,}000(7.5) + 50{,}000 \\\\ \amp= -200(56.25) + 22{,}500 + 50{,}000 \\\\ \amp= -11{,}250 + 22{,}500 + 50{,}000 \\\\ \amp= 61{,}250 \end{align*}
    Thus, the maximum revenue is \(\$61{,}250\text{,}\) which occurs when \(x = 7.5.\text{.}\) To find the price that maximizes revenue, we substitute \(x = 7.5\) into the price equation:
    \begin{align*} p\amp= 1 + 0.1(7.5) \\\\ \amp= 1 + 0.75 \\\\ \amp= 1.75 \end{align*}
    Therefore, the price that maximizes revenue is \(\$1.75\text{.}\) To find the number of downloads, we perform the following calculation:
    \begin{align*} n\amp= 50{,}000 - 2{,}000(7.5) \\\\ \amp= 50{,}000 - 15{,}000 \\\\ \amp= 35{,}000 \end{align*}
    Therefore, the number of downloads at that price is approximately \(35{,}000.\)
