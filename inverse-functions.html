    Section 7.13 Inverse Functions

    Definition 7.13.1. Definition.

    Let \(f(x)\) be any function and let \((x,f(x))\) be any point on its graph. Then the inverse of \(f(x)\) is the set of all points of the form \((f(x),x)\text{.}\)

    Note 7.13.2. Important Note.

    The inverse of a function is rarely a function itself.

    Example 7.13.3. Example 3.

    Show that the inverse of \(f(x)=x^2\) is>not a function.
    Solution.
    Lets consider some sample points from the graph of \(f(x)=x^2\text{:}\)
    \begin{equation*} (-2,4), (-1,1), (0,0), (1,1), (2,4) \end{equation*}
    By definition, the inverse has the sample points
    \begin{equation*} (4,-2), (1,-1), (0,0), (1,1), (4,2) \end{equation*}
    While its not explicitly stated, since we are just switching the coordinates, the graph of the inverse keeps the same shape as the original function but will be oriented in a different directed. Plotting the points for the inverse and drawing a parabola through them, we get the following graph.
    Since the inverse graph fails the vertical line test, the inverse is not a function.

    Theorem 7.13.5. Theorem.

    Any function that passes the horizontal line test has an inverse that is also a function.

    Notation 7.13.6. Notation for an Inverse Function.

    If the inverse of \(f(x)\) is a function, we denote it by \(f^{-1}(x)\text{.}\)

    Note 7.13.7. Warning.

    \(f^{-1}(x)\ne \dfrac{1}{f(x)}\text{.}\) If we want the reciprocal, we write \([f(x)]^{-1}=\dfrac{1}{f(x)}\text{.}\)

    Example 7.13.8. Example 4.

    Show that \(f(x)=x^2,\; x\ge 0\) has an inverse function. Graph the inverse function, and identity the inverse function from its graph.
    Solution.
    First, lets graph the function \(f(x)\text{.}\)
    Since the modified graph of \(x^2\) passes the horizontal line test, the inverse for \(f(x)\) is a function.
    To graph the inverse, we start with some sample points for \(f(x)\text{:}\)
    \begin{equation*} (0,0), (1,1), (2,4), (3,9) \end{equation*}
    By definition, the inverse will have the points
    \begin{equation*} (0,0), (1,1), (4,2), (9,3) \end{equation*}
    Plotting these points and drawing a half-parabola through them yields the following graph.
    From the graph, we can see that the inverse function is the square root function,
    \begin{equation*} y=\sqrt{x} \end{equation*}
    Since the square root function is an inverse function for \(f(x)\text{,}\) we write
    \begin{equation*} f^{-1}(x)=\sqrt{x} \end{equation*}
    instead of \(y=\sqrt{x}\text{.}\)

    Theorem 7.13.11. Inverse Function Property.

    Let \(f(x)\) be any one-to-one function. Then the inverse function \(f^{-1}(x)\) exists and has the following properties.
    1. \((f^{-1}\circ f)(x)=x\) for any \(x\) in the domain of \(f\text{.}\)
    2. \((f\circ f^{-1})(x)=x\) for any \(x\) in the range of \(f\text{.}\)
    3. The domain of \(f^{-1}\) equals the range of \(f\text{.}\)
    4. The range of \(f^{-1}\) equals the domain of \(f\text{.}\)

    Example 7.13.12. Example 9.

    Determine if \(f(x)=x^2\) and \(g(x)=\sqrt{x}\) are inverse functions.
    Solution.
    Even though
    \begin{align*} (f\circ g)(x) \amp=f(g(x)\\) \\ \amp=f(\sqrt{x})\\\\ \amp=(\sqrt{x})^2\\\\ \amp=x\\ \end{align*}
    and
    \begin{align*} (g\circ f)(x) \amp=g(f(x)\\) \\ \amp=g(x^2)\\\\ \amp=\sqrt{x^2}\\\\ \amp=x\\ \end{align*}
    since the range of \(g(x)=\sqrt{x}\) is \([0,\infty)\) and the domain of \(f(x)=x^2\) is \((-\infty,\infty)\text{,}\) the two functions are not inverses of each other.

    Example 7.13.13. Example 10.

    Determine if \(f(x)=3x-2\) and \(g(x)=\dfrac{x+2}{3}\) are inverse functions.
    Solution.
    Since
    \begin{align*} (f\circ g)(x) \amp=f(g(x)\\) \\ \amp=f\left(\dfrac{x+2}{3}\right)\\\\ \amp=3\left(\dfrac{x+2}{3}\right)-2\\\\ \amp=x+2-2\\\\ \amp=x\\ \end{align*}
    and
    \begin{align*} (g\circ f)(x) \amp=g(f(x)\\) \\ \amp=g(3x-2)\\\\ \amp=\dfrac{(3x-2)+2}{3}\\\\ \amp=\dfrac{3x}{3}\\\\ \amp=x\\ \end{align*}
    and since both functions have domain and range of \((-\infty,\infty)\text{,}\) the two functions are inverses of each other.

    Procedure 7.13.14. Finding the Inverse of a Function Algebraically.

    1. Replace \(f(x)\) with \(y\text{.}\)
    2. Interchange \(x\) and \(y\text{.}\)
    3. Solve for \(y\text{.}\)
    4. Replace \(y\) with \(f^{-1}(x)\text{.}\)

    Example 7.13.15. Example 11.

    Find the inverse for \(f(x)=5x+3\text{.}\)
    Answer.
    \(f^{-1}(x)=\dfrac{x-3}{5}\text{.}\)
    Solution.
    We will follow the procedure outlined above to produce the following calculation:
    \begin{align*} y\amp=5x+3\\\\ x\amp=5y+3\\\\ x-3\amp=5y\\\\ y\amp=\dfrac{x-3}{5}\\\\ f^{-1}(x)\amp=\dfrac{x-3}{5}\\ \end{align*}
    Therefore, the inverse function is \(f^{-1}(x)=\dfrac{x-3}{5}\text{.}\)

    Example 7.13.16. Example 12.

    Find the inverse for \(f(x)=\dfrac{3-x}{x+9}\text{.}\)
    Answer.
    \(f^{-1}(x)=\dfrac{3 - 9x}{x + 1}\text{.}\)
    Solution.
    We will follow the procedure outlined above to produce the following calculation:
    \begin{align*} y\amp=\dfrac{3-x}{x+9}\\\\ x(y+9)\amp=3 - y\\\\ xy + 9x\amp=3 - y\\\\ xy + y\amp=3 - 9x\\\\ y(x + 1)\amp=3 - 9x\\\\ y\amp=\dfrac{3 - 9x}{x + 1}\\\\ f^{-1}(x)\amp=\dfrac{3 - 9x}{x + 1}\\ \end{align*}
    Therefore, the inverse function is \(f^{-1}(x)=\dfrac{3 - 9x}{x + 1}\text{.}\)

    Theorem 7.13.17. Graphs of Inverse Functions.

    If the point \((a,b)\) is on the graph of \(y=f(x)\text{,}\) then the point \((b,a)\) is on the graph of \(y=f^{-1}(x)\text{.}\)

    Example 7.13.18. Example 14.

    The graph of a one-to-one function \(f(x)\) with some sample points is given. Sketch the graph of \(f^{-1}(x)\text{.}\)
    Solution.
    Since the sample points are
    \begin{equation*} \left(-2,\frac{1}{4}\right),\left(-1,\frac{1}{2}\right), (0,1), (1,2), (2,4)\text{,} \end{equation*}
    the corresponding points on the graph of the inverse function will be
    \begin{equation*} \left(\frac{1}{4},-2\right),\left(\frac{1}{2},-1\right), (1,0), (2,1), (4,2)\text{.} \end{equation*}
    Plotting these points and drawing a smooth curve through them yields the following graph.
    Notice that neither of these graphs are ones we are familiar with, but yet with enough information form one of the graphs, we can still find the graph of its inverse. We will be studying these functions in the next chapter.

    Procedure 7.13.21. Creating Inverses by restricting a function.

    Let \(f(x)\) be a function that is not one-to-one. If we restrict the domain of \(f(x)\) so that it is one-to-one on that domain, then the inverse of the restricted function will be a function.

    Example 7.13.22. Example 15.

    The function \(f(x)=\dfrac{1}{(x+2)^2}\) is not one-to-one since its graph does not pass the horizontal line test Find a restriction of the domain that will make \(f(x)\) one-to-one. Then find the inverse of the restricted function.
    Solution.
    First, lets graph our function to see what the graph looks like.
    Notice that our graph is a reciprocal function with a vertical asymptote at \(x=-2\) and a horizontal asymptote at \(y=0\text{.}\) To make this function one-to-one, we can restrict the domain to \(x>-2\) or \(x<-2\text{.}\) Lets choose \(x>-2\) for our restriction. Now, lets find the inverse of the restricted function.
    \begin{align*} y\amp=\dfrac{1}{(x+2)^2}\amp x\amp> -2\\\\ x\amp=\dfrac{1}{(y+2)^2}\amp y\amp> -2\\\\ (y+2)^2\amp=\dfrac{1}{x}\amp y\amp> -2\\\\ y+2\amp=\sqrt{\dfrac{1}{x}}\amp y\amp> -2\\\\ y\amp=\pm\sqrt{\dfrac{1}{x}} - 2\amp y\amp> -2\\ \end{align*}
    The question becomes: do we choose the positive or negative root? Since \(y>-2\text{,}\) this implies that
    \begin{equation*} \pm\sqrt{1}{x}-2>-2 \end{equation*}
    Solving for the square root, we get that
    \begin{equation*} \pm\sqrt{\dfrac{1}{x}}>0 \end{equation*}
    This equation is only true if we choose the positive square root. Therefore, the inverse function is
    \begin{equation*} f^{-1}(x)=\sqrt{\dfrac{1}{x}} - 2,\; x>0 \end{equation*}
