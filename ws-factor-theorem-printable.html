    Procedure A.5.13. Graphing Polynomial Functions.

    Let \(p(x) = a_n x^m + a_{n-1} x^{n-1} + \ldots + a_2 x^2 + a_1 x + a_0\) be any polynomial function. Use the following steps to sketch a graph by hand.
    1. Determine the end behavior.
    2. Find the \(y\)-intercept.
    3. Determine the zeroes (if they exist) and their multiplicities.
    4. Plot additional points as necessary.
    5. Make sure there are no more than \(n - 1\) turning points (local extrema).

    1. Graphing a Degree 3 Polynomial.

    Graph the function \(p(x) = x^3 + 2x^2 - 4x - 8\text{.}\)
    Consider the following graph of \(p(x)=x^3-3x-2\text{.}\)
    While it is clear from the graph that we have two \(x\)-intercepts: \((-1,0)\) and \((2,0)\text{,}\) try to solve \(x^3-3x-2=0\) using any of the techniques we have learned so far. You will see that they all fail. Furthermore, the polynomial factors nicely as \(p(x)=(x-2)(x+1)^2\text{.}\) Again, no factoring technique we have learned will factor this polynomial.

    Theorem A.5.15. The Division Algorithm for Natural Numbers.

    Let \(p\) and \(d\) be natural numbers (nonzero, positive, whole numbers). Then there exist two unique numbers, a quotient \(q\) and a remainder \(r\text{,}\) such that
    \begin{equation*} p = q\cdot d + r\text{,} \end{equation*}
    where \(r < d\text{.}\)
    • \(p\) is known as the dividend.
    • \(d\) is known as the divisor.
    The Division Algorithm gives us a way to define division in terms of multiplication and addition for any two numbers \(p\) and \(d\text{.}\) We find the values \(q\) and \(r\) using long division. Our next two examples show us what happens.

    2. Long Division with a Zero Remainder.

    Use long division to divide \(136\) by \(4\text{.}\) Then express the result using the Division Algorithm.

    Definition A.5.16. The Zero Polynomial.

    The zero polynomial is the polynomial whose value is zero for every value of \(x\text{.}\) We write the zero polynomial as
    \begin{equation*} p(x) = 0 \end{equation*}
    for all real numbers \(x\text{.}\)

    Note A.5.17.

    Dont confuse the zero polynomial \(p(x) = 0\) with solving a polynomial equation \(p(x) = 0\text{.}\) While the notations are identical, the contexts are very different. Thankfully, these concepts dont typically come into direct contact with one another, so the context of the problem will tell you which version of the notation we are using.

    Theorem A.5.18. The Division Algorithm.

    Let \(p(x)\) and \(d(x)\) be polynomials, where \(d(x)\) is not the zero polynomial. Then there exist unique polynomials, a quotient \(q(x)\) and a remainder \(r(x)\text{,}\) such that
    \begin{equation*} p(x) = q(x) d(x) + r(x) \end{equation*}
    where the degree of \(r(x)\) is less than the degree of \(d(x)\text{.}\)
    • \(p(x)\) is called the dividend.
    • \(d(x)\) is called the divisor.

    3. Polynomial Long Division Example.

    Use polynomial long division to divide \(p(x) = x^2 + 3x - 1\) by \(d(x) = x + 4\text{.}\) Then express the result using the Division Algorithm for polynomials.

    4. Polynomial Long Division Example.

    Use polynomial long division to divide \(p(x) = x^4 - x^2 - 2x - 1\) by \(d(x) = x^2 - x - 1\text{.}\) Then express the result using the Division Algorithm for polynomials.
    Recall that we are looking for a way to factor any polynomial function as
    \begin{equation*} p(x)=a_n(x-c_1)(x-c_2)\cdot(x-c_{n-1})(x-c_n)\text{.} \end{equation*}
    Since we have seen that there is a relationship between factoring and dividing, notice that all our factors look like \(x\) minus a number. So, lets look at how dividing by \(x-c\) and the Division Algorithm work together.
    But polynomial long division, as we have seen, can be long, tedious, and cumbersome to write. It turns out that when we divide by \(x-c\text{,}\) there is a shorthand version of polynomial long division called synthetic division that can make this process faster and less error-prone. Instead of writing a step-by-step method, we will just illustrate how this works in the next three examples.

    5. Synthetic Division, First Example.

    Use Synthetic Division to divide \(p(x)=x^3+8x^2+13x+3\) by \(d(x)=x+3\text{.}\) Write your final answer using the Division Algorithm.

    6. Synthetic Division, Second Example.

    Use Synthetic Division to divide \(p(x)=2x^3-9x^2-5x+36\) by \(d(x)=x-4\text{.}\) Write your final answer using the Division Algorithm.

    7. Synthetic Division, Third Example.

    Use Synthetic Division to divide \(p(x)=x^4+2x^3+x+2\) by \(d(x)=x+2\text{.}\) Write your final answer using the Division Algorithm.

    Theorem A.5.19. The Remainder Theorem.

    Let \(p(x)\) be any polynomial function and let \(d(x) = x - c\) for some real number \(c\text{.}\) Then the remainder is equal to \(p(c)\text{.}\)

    8. Evaluating a Polynomial Using the Remainder Theorem.

    Let \(p(x) = 3x^3 - x^2 + 92x + 4\text{.}\) Evaluate \(p(-5)\text{.}\)

    Theorem A.5.20. The Factor Theorem.

    Let \(p(x)\) be any polynomial function. The following two statements are equivalent to each other.
    1. \(x=c\) is a zero of \(p(x)\text{.}\)
    2. \(x-c\) is a factor of \(p(x)\text{.}\)
    Since we just proved that factoring is another way to find solutions to polynomial equations,
    1. We dont have to set polynomials equal to zero to solve them; we just need to factor them. This alleviates the confusion between solving an equation and simplifying an expression.
    2. If we know a zero of the polynomial, we can use Synthetic or Polynomial Long Division to find the other factor that doesnt correspond to the zero. This helps us factor higher-degree polynomials.

    9. Factoring a Degree 4 Polynomial.

    Suppose you know that \(x = -3\) and \(x = 4\) are zeroes of the polynomial
    \begin{equation*} p(x) = x^4 + 2x^3 - 25x^2 - 26x + 120\text{.} \end{equation*}
    Find the other zeroes of the polynomial.
    The overall technique we will use is
    1. Find any zero of \(p(x)\text{.}\)
    2. Use synthetic division to factor the polynomial.
    3. Repeat steps 1 and 2 until you end with a quadratic factor.
    4. Use the methods from Chapter 4 to find the remaining two solutions (if they exist).
