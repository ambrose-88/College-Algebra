    Worksheet A.8.1 Exponential and Logarithmic Functions
    

    Definition A.8.1.

    An exponential function is any function of the form \(f(x)=a^x\text{,}\) where \(a>0\) is a positive real number not equal to 1.

    1. Evaluating Exponential Functions.

    Evaluate the following functions.
    • \(f(x)=2^x\) at \(x=3\)
    • \(g(x)=3^{-x}\) at \(x=2\)
    • \(h(x)=\left(\dfrac{3}{2}\right)^x\) at \(x=4\)
    • \(m(x)=\left(\dfrac{1}{4}\right)^x\) at \(x=-2\)
    • \(k(x)=5^x\) at \(x=\frac{3}{2}\)
    • \(f(x)=7^{x}\) at \(x=\pi\)

    Theorem A.8.2. Exponential Functions with \(a > 1\).

    Let \(f(x) = a^x\) be an exponential function where \(a > 1\text{.}\) The graph of \(f\) has the following properties:
    • Domain: \(\left( -\infty, \infty \right)\)
    • Range: \(\left( 0, \infty \right)\)
    • \(y\)-intercept: \((0, 1)\text{.}\)
    • End Behavior:
      • As \(x \to \infty, f(x) \to \infty\)
      • As \(x \to -\infty, f(x) \to 0\text{.}\) This implies the \(x\)-axis is a horizontal asymptote.
    • The function is increasing on \(\left( -\infty, \infty \right)\text{.}\)

    2. Graphing an Exponential Function with \(a > 1\).

    Graph \(f(x) = 2^x\text{.}\)

    Theorem A.8.4. Exponential Functions with \(0 < a < 1\).

    Let \(f(x) = a^x\) be an exponential function where \(0 < a < 1\text{.}\) The graph of \(f\) has the following properties:
    • Domain: \(\left( -\infty, \infty \right)\)
    • Range: \(\left( 0, \infty \right)\)
    • \(y\)-intercept: \((0, 1)\text{.}\)
    • End Behavior:
      • As \(x \to \infty, f(x) \to 0\text{.}\) This implies the \(x\)-axis is a horizontal asymptote.
      • As \(x \to -\infty, f(x) \to \infty\)
    • The function is decreasing on \(\left( -\infty, \infty \right)\text{.}\)

    3. Graphing an Exponential Function with \(0 < a < 1\).

    Graph \(f(x) = \left( \dfrac{1}{2} \right)^x\text{.}\)

    Definition A.8.6.

    • The mathematical constant \(e\) is an irrational number approximately equal to \(2.71828\text{.}\)
    • The function \(\exp(x) = e^x\) is called the natural exponential function.

    Example A.8.8. Evaluating \(y=e^x\).

    Use a calculator to evaluate the following.
    • \(f(x) = e^x\) at \(x=3\)
    • \(g(x) = e^x\) at \(x=-2\)
    • \(y = \exp(x)\) at \(x=\frac{1}{2}\)
    • \(y=\exp(x)\) at \(x=\pi\)

    Example A.8.9. How to Solve for a Variable in the Exponent using graphs.

    Let \(f(x) = 3^x\text{.}\) Solve the equation \(f(x)=9\) using a graph.

    Definition A.8.10.

    Let \(f(x)=b^x\) be an exponential function with \(b>0\) and \(b\neq 1\text{.}\) The inverse function of \(f\) is called the logarithmic function with base \(b\) and is denoted by \(y=\log_b(x)\text{.}\) This means that \(y=\log_b(x)\) if and only if \(b^y=x\text{.}\) In other words, \(log_b(x)\) is the exponent to which we must raise \(b\) to obtain the value \(x\text{.}\)

    Note A.8.11. Parentheses in Logarithmic Expressions.

    • When you take a logarithm of a single term, it is common practice to drop the parentheses like this: \(\log_b(x)=\log_b x\text{.}\) I will never drop the parentheses in my lecture notes.
    • Taking the logarithm of an expression with more than one term always requires parentheses: \(\log_b(x+1) \neq \log_b x + 1 = \log_b(x) + 1\text{.}\)

    Notation A.8.12.

    Two logarithmic functions are given special notation.
    • The logarithmic function with base 10 is called the common logarithm and is denoted by \(\log(x)\text{.}\) In other words, \(\log(x)=\log_{10}(x)\text{.}\)
    • The logarithmic function with base \(e\) is called the natural logarithm and is denoted by \(\ln(x)\text{.}\) In other words, \(\ln(x)=\log_{e}(x)\text{.}\)

    Example A.8.13. Converting Logarithmic to Exponential Form.

    Convert each logarithmic equation to its equivalent exponential form.
    • \(\displaystyle \log_3(27)=3\)
    • \(\displaystyle \log(1000)=3\)
    • \(\displaystyle \ln(22)=y\)

    Example A.8.14. Converting Exponential to Logarithmic Form.

    Convert each exponential equation to its equivalent logarithmic form.
    • \(\displaystyle 4^3=64\)
    • \(\displaystyle 10^{-2}=0.01\)
    • \(\displaystyle e^x=12\)
